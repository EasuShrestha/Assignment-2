Qs7. Differentiate Inline and Block Elements:
• Task: Create a heading using the h1 tag and some text using the span tag. 
Describe the difference between inline and block elements.
  
Difference between inline and block elements.

  Block elements cannot be directly used inside a paragraph because they would break the flow of the paragraph.
    If you try to put a block element inside a paragraph, it will usually break the paragraph into separate elements.
    Meanwhile,
    Inline elements can be used directly within a paragraph without breaking the flow of text.
They integrate seamlessly within the paragraph, allowing for formatting and linking text without disrupting the text flow.
